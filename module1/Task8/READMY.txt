Задание 8
Установите web приложение на Tomcat сервер. Настройте контекст приложения
таким образом, чтобы war файл автоматически распаковывался при запуске сервером
приложений. Проверьте работоспособность вашего приложения.

Для того, чтобы web приложение установить на Tomcat сервер нужно war архив скопировать в базовую директорию для приложений
(у меня CATALINA_HOME/webapps). Чтобы war распаковывался автоматически при запуске сервером приложений необходимо зайти в
в файл server.xml, который находиться в CATALINA_HOME/conf/server.xml и найти настройки хоста:
<Host name="localhost" appBase="webapps" unpackWARs="true" autoDeploy="true"> где если unpackWARs="true" будет происходить
автоматическое распаковывание web приложение при запуске сервера.
