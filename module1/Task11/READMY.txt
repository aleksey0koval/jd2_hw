Задание 11
Скопируйте веб-проект в хранилище, созданное ранее. Затем скачайте (checkout)
изменения в отдельный каталог. Попробуйте изменить что-либо в вашем проекте
и посмотрите на статус папок в проводнике Windows.

Чтобы скопировать проект в хранилище нужно сначало эту папку с проектом указать гиту для отслеживания комнадой
git init. Затем дабавить ресурсы для отслеживания командой git add <fileName> либо все при помощи "." git add .
После этого зафиксировать изменения в локольном репозитории командой git commint -m "aaaaa". После этого запушиваем
изменения в удаленный репозиторий командой git push origin nameBranch. Если адрес удаленного репозитория неизвестен,
то командой git remote add origin https://github.com/nameProf/nameReposit.git указываем его и затем его запушиваем.
Создаем новую ветку newMaster командой git checkout -b newMaster и сразу переходим на нее. Производим изменения
файле READMY.txt и проверяем статус командой git status и проверяем статус командой git status: в ветке newMaster
