Задание 4
Выполните сборку проекта с различными фазами жизненного цикла сборки.

Существует три встроенных жизненных цикла сборки: стандартный, очистить и сайт.
Стандартный цикл обрабатывает компиляцию проекта, тестирование и установка (validate, compile, test, integtation-test, package, verify, install, deploy).
Цикл очистки овтечает за процесс очистки проекта от предыдущих файлов сборки (clean). 
Цикл сайт отвечает за генерацию и установку документации проекта.
Сборку проекта можно производить как из командной строки, так и из среды разработки.
При работе из командной строки можно пользоваться следующими коминациями:
mvn clean copmile package install либо по отдельность. Если по отдельности, то верняя оперция содержить нижнюю(в install входить package и compile)
